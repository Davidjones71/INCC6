PSA Operator-Inspector Measurement Database and Logsheet File Format

All values will be separated by commas. If a text string contains a space or other white space character, it will be enclosed in double quotes. All fields will always be present. NULL strings (“”) or zeros will be used when a field is not applicable. All floating point values will have six digits to the right of the decimal point.

The default file name is NCC_PSA.dat
