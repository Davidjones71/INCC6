Operator Declaration File Format for Curium Ratio Measurements

All values within a record must be separated by commas and there must not be any carriage return and line feed (CRLF). If a text string contains a space, tab or comma , it must be enclosed in double quotes. All fields must always be present. Each record must be terminated by a carriage return and line feed (CRLF), but no comma. The exception is the last record which must not have either a comma or CRLF terminator.

Operator declaration files are entirely in text (ASCII).

Record format:

Item id                                       12 characters

Label for id                                 12 characters

Id                                             12 characters

Input batch id                              12 characters

Declared Pu mass (g)                   floating point

Declared U mass (g)                     floating point

Declared 235U mass (g)                  floating point

Cm/Pu ratio                                floating point

Cm/Pu ratio error                        floating point

Cm/Pu ratio date                         YYYYMMDD

Cm/U ratio                                  floating point

Cm/U ratio error                          floating point

Cm/U ratio date                           YYYYMMDD
