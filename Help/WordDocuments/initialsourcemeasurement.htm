INCC Analysis Algorithms

Initial Source Measurement

The initial source measurement determines the doubles rate and error for a reference 252Cf source. These values and the measurement date are saved automatically and are used for the analysis of future normalization measurements. The reference 252Cf source must be measured long enough to reach a precision for the doubles rate of 0.3 % (default) or the results are not accepted.
