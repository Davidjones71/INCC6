Material Type Add/Delete

To use this feature, select Maintain | Material Type Add/Delete. A default material type of Pu is provided with INCC when it is installed.

To add a new material type, select “Add material type” from the dialog box, and then a dialog box will be displayed for you to type in the new material type (maximum of 5 characters) which will be placed in the material type list. There is a maximum of 100 material types.

To delete an material type, select “Delete material type” from the dialog box, and then a dialog box will be displayed for you to select the material type to be deleted from a drop down list.
