Introduction

This Document's Conventions

1.  Italics are used to denote menu selections to be made by the user, for example, select Measurement Parameters .

2.  A sequence of operations is indicated by using a vertical bar to separate the individual operations, for example, select File | Exit INCC means select the File menu and then the Exit option from that menu.

3.  File names and file path names are specified with the Courier New font, for example, c:\INCC\data.

4.  Character strings displayed on the screen are surrounded by double quotes, for example, you would see the message "Warning - no analysis methods selected for calibration." if you attempted run a calibration measurement without first defining the analysis method for the specified detector.

5.  Square brackets are used to denote single keystrokes that cannot be represented by a single character, for example, [Enter].

6.  Warnings for functions that may inadvertently cause a loss of data or parameters are in boldface type.

7.  By convention, the INCC installation location will be referred to as \INCC in this document.
