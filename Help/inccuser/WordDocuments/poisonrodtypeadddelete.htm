Poison Rod Type Add/Delete

To use this feature, select Maintain | Poison Rod Type Add/Delete. A default poison rod type of G for gadolinium is provided with INCC when it is installed.

To add a new poison type, select “Add poison rod type” from the dialog box, and then a dialog box will be displayed for you to type in the new poison rod type (maximum of one character) and its poison absorption factor (default is .647) which will be placed in the poison rod type list. There is a maximum of 10 poison types.

 

The default poison absorption factor for each rod type may be overridden for select Detector, Material Type and Thermal Code settings in Collar Calibration parameters. See the description in the Collar Calibration section.

To delete a poison rod type, select “Delete poison rod type” from the dialog box, and then a dialog box will be displayed for you to select the poison rod type to be deleted from a drop down list.
